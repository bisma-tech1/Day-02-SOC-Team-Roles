# 🛡️ Day 02 - SOC Team Roles (L1, L2, L3, IR)

## 🚀 What I Learned Today

Today I learned that a Security Operations Center (SOC) works as a team with different roles.

## 👨‍💻 SOC Roles

### L1 Analyst
- Monitors alerts
- Reviews suspicious activity
- Performs initial investigation

### L2 Analyst
- Handles serious incidents
- Performs deeper analysis
- Investigates malware or compromised systems

### L3 Analyst
- Advanced level analyst
- Creates detection rules
- Performs threat hunting

### IR Team (Incident Response)
- Responds during cyber attacks
- Controls damage
- Helps recover affected systems

## 📌 Summary

A SOC is not one person. It is a complete team working together to protect an organization.
